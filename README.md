Description
-----------------------------------------------
Analyse de données du baccalauréat 2019 par académie.



Dataset
-----------------------------------------------

- Provider: Ministère de l'Education Nationale et de la Jeunesse
- Dataset: _fr-en-baccalaureat-par-academie.csv_
- Source: [www.data.gouv.fr](www.data.gouv.fr)



Installation
------------------------------------------------
To install the current version:

```shell
git clone git@github.com:pmatran/bac.git
cd bac
```
